# U-Net

Implementation of the [U-Net model](https://arxiv.org/abs/1505.04597).

## Model Details

For benchmarking on the Well, we used the following parameters.

| Parameters          | Values |
|---------------------|--------|
| Spatial Filter Size | 3      |
| Initial Dimension   | 48     |
| Block per Stage     | 1      |
| Up/Down Blocks      | 4      |
| Bottleneck Blocks   | 1      |
